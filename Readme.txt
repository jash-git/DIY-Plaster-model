調製石膏漿 [石膏粉比例]

資料來源: https://beautydiy.pixnet.net/blog/post/31521104-%E5%A4%B1%E6%95%97%E7%9A%84%E7%9F%B3%E8%86%8F%E6%A8%A1

製作鑄漿模的石膏稠度約80，即石膏粉和水的比例為100：80，即石膏粉重一公斤，則水的重量為0.8公斤。
1.將水桶裝預定的水量置於石膏秤上，扣除桶子的重量，得水的重量。
2.另以磅秤量取水重*1.2重量的石膏粉
3.將秤好的石膏粉均勻的撒入水桶內
4.石膏粉完全撒入水桶內，先不要攪拌應靜置 1- 2 分鐘，待石膏粉完全浸漬後再行攪拌。
5.以手或攪拌器依順時針方向攪拌，每分鐘約 100 -150 圈。攪拌時，石膏漿內氣泡會浮出水面，並自動消失。
6.攪拌約5分鐘，石膏漿內的氣泡消除 ，並可感些微阻力，此時完成攪拌石膏，可預備澆鑄。
